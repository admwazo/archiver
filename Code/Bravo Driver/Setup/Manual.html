Follow these instructions to manually install the driver after a failure of the automatic method.

1. To install the driver Click the Windows button - Right Click on Computer - Click on Manage.

2. Click on Device Manager - Find the Primera product name under Other Devices - Right Click on it - Choose Update Driver.

3. Click Browse my computer...

4. Browse to the location of the driver. The driver automaticaly extracts to the root of your C drive in a folder named for the product (Example: C:\DPProDriver\). Click Next.

5. Click "Install this driver software anyway."

6. You will receive a "Windows has successfully updated..." message when complete.
